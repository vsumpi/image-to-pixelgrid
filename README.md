# Usage
1. Get a picture of a **pixel art**.

**NOTE:**
  - Transparent images won't work yet!
  - Only submit **pixel arts**!
  - Normal images will result in blank images!
2. Drag & Drop to the console
3. Press Enter and open the file named "_out.jpg_"
